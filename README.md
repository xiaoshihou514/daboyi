本项目是欧陆风云5架空地图编辑器，允许玩家创建、编辑和定制自己的架空世界地图。支持自定义国家内行政区（参考十字军之王3）。

提供linux x86整合包，其他操作系统可以自行编译或使用[网页版](https://xiaoshihou514.github.io/daboyi)。

本项目是纯AI开发的项目，如果想要贡献或二次开发可以参考`doc/`的面向AI文档。
