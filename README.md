# 大博弈

欧陆风云5架空地图编辑器

[启动！](https://xiaoshihou514.github.io/daboyi)
</div>

本项目是欧陆风云5架空地图编辑器，允许玩家创建、编辑和定制自己的架空世界地图。支持自定义国家内行政区（参考十字军之王3）。
