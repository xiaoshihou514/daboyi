# 涂色功能技术计划书

## 1. 功能概述

实现一个完整的地图涂色编辑器，支持：
- **国家管理**：添加/删除国家，设置国家颜色和首都
- **行政区系统**：国家下属行政区，支持无限层级嵌套
- **省份分配**：将省份添加到国家或行政区
- **刷子工具**：可调整大小的画笔，快速涂色相邻省份
- **丝滑操作**：异步处理，不阻塞 UI 渲染
- **撤销/重做**：支持操作历史回溯

**实现状态**: ✅ 核心功能已完成 (2026-03-26)

### 4.1 刷子工具 (`client/src/brush.rs`)

```rust
use bevy::prelude::*;
use crate::map::{MapResource, MAP_WIDTH};
use crate::editor::{BrushTool, DragState, AdminMap, CountryMap, ActiveAdmin, ActiveCountry};
// 基础数值转换直接使用 `as`

/// 刷子输入处理系统
pub fn brush_input_system(
    mouse_input: Res<ButtonInput<MouseButton>>,
    mouse_pos: Res<CursorPos>,
    camera_q: Query<(&Camera, &GlobalTransform), With<Camera2d>>,
    map: Res<MapResource>,
    mut brush: ResMut<BrushTool>,
    mut drag: ResMut<DragState>,
    mut admin_map: ResMut<AdminMap>,
    country_map: Res<CountryMap>,
    active_admin: Res<ActiveAdmin>,
    active_country: Res<ActiveCountry>,
    mut recolor_pending: Local<bool>,
) {
    // 切换刷子：B 键
    if mouse_input.just_pressed(MouseButton::Middle) {
        brush.enabled = !brush.enabled;
        return;
    }

    // 调整刷子大小：滚轮
    if let Some(scroll) = mouse_input.get_scroll() {
        brush.radius = (brush.radius + scroll.y * 0.5).clamp(0.5, 10.0);
        return;
    }

    if !brush.enabled {
        drag.is_dragging = false;
        return;
    }

    // 获取鼠标世界坐标
    let Some(world_pos) = get_mouse_world_pos(&mouse_pos, &camera_q) else { return };

    // 左键按下开始拖拽
    if mouse_input.just_pressed(MouseButton::Left) {
        drag.is_dragging = true;
    }
    if mouse_input.just_released(MouseButton::Left) {
        drag.is_dragging = false;
        drag.last_province = None;
    }

    if !drag.is_dragging {
        return;
    }

    // 查找鼠标下的所有省份（考虑刷子半径）
    let provinces_in_brush = find_provinces_in_radius(world_pos, brush.radius, &map);

    // 获取目标行政区
    let target_admin = active_admin.0.or_else(|| {
        // 如果没有选中行政区，但选中了国家，尝试找到该国家的顶级行政区
        active_country.0.as_ref().and_then(|tag| {
            // 这里可以查找该国家的第一个行政区
            None
        })
    });

    // 分配省份到目标
    for &prov_id in &provinces_in_brush {
        if drag.last_province == Some(prov_id) {
            continue; // 跳过已处理的省份
        }

        if let Some(admin_id) = target_admin {
            admin_map.0.insert(prov_id, admin_id);
        } else if let Some(country_tag) = &active_country.0 {
            // 直接分配到国家（清除行政区归属）
            admin_map.0.remove(&prov_id);
            country_map.0.insert(prov_id, country_tag.clone());
        }

        *recolor_pending = true;
    }

    if !provinces_in_brush.is_empty() {
        drag.last_province = provinces_in_brush.first().copied();
    }
}

/// 获取鼠标世界坐标
fn get_mouse_world_pos(
    mouse_pos: &Res<CursorPos>,
    camera_q: &Query<(&Camera, &GlobalTransform), With<Camera2d>>,
) -> Option<[f32; 2]> {
    let Ok((camera, cam_transform)) = camera_q.get_single() else { return None };
    let cursor_pos = mouse_pos.0?;
    let world_pos = camera.viewport_to_world_2d(cam_transform, cursor_pos).ok()?;
    Some([world_pos.x, world_pos.y])
}

/// 查找半径内的所有省份
fn find_provinces_in_radius(
    pos: [f32; 2],
    radius: f32,
    map: &MapResource,
) -> Vec<ProvinceId> {
    let mut result = Vec::new();
    let radius_sq = radius * radius;

    for prov in &map.0.provinces {
        if prov.centroid[0] == 0.0 && prov.centroid[1] == 0.0 {
            continue;
        }
        let dx = prov.centroid[0] - pos[0];
        let dy = prov.centroid[1] - pos[1];
        if dx * dx + dy * dy <= radius_sq {
            result.push(prov.id);
        }
    }

    result
}

/// 刷子 UI 渲染（使用 Bevy UI 或 egui）
pub fn brush_ui_system(
    brush: Res<BrushTool>,
    mut gizmos: Gizmos,
) {
    if !brush.enabled {
        return;
    }

    // 绘制刷子范围圆圈
    // 注意：需要获取当前鼠标位置
    // 这里使用简化的示意代码
    gizmos.circle_2d(
        /* center */ [0.0, 0.0],
        /* radius */ brush.radius,
        /* color */ Color::srgba(1.0, 1.0, 1.0, 0.5),
    );
}
```

```rust
/// 验证行政区分配：确保省份只能属于父行政区的省份集合
pub fn validate_admin_assignments(
    admin_areas: Res<AdminAreas>,
    mut admin_map: ResMut<AdminMap>,
) {
    // 构建行政区层级映射
    let admin_children: std::collections::HashMap<AdminId, Vec<AdminId>> = {
        let mut map = std::collections::HashMap::new();
        for area in &admin_areas.0 {
            if let Some(parent_id) = area.parent_id {
                map.entry(parent_id).or_insert_with(Vec::new).push(area.id);
            }
        }
        map
    };

    // 验证每个省份的行政区归属
    let mut to_remove = Vec::new();
    for (&prov_id, &admin_id) in admin_map.0.iter() {
        if !is_valid_assignment(prov_id, admin_id, &admin_areas.0, &admin_map.0, &admin_children) {
            to_remove.push(prov_id);
        }
    }

    for prov_id in to_remove {
        admin_map.0.remove(&prov_id);
    }
}

/// 检查省份分配是否有效
fn is_valid_assignment(
    _province_id: ProvinceId,
    admin_id: AdminId,
    admin_areas: &[AdminArea],
    _admin_map: &HashMap<ProvinceId, AdminId>,
    _admin_children: &std::collections::HashMap<AdminId, Vec<AdminId>>,
) -> bool {
    // 查找目标行政区
    let Some(target_admin) = admin_areas.iter().find(|a| a.id == admin_id) else {
        return false;
    };

    // 如果有父行政区，需要确保逻辑一致
    // （这里可以根据具体需求添加更复杂的验证）
    if let Some(parent_id) = target_admin.parent_id {
        // 父行政区必须存在
        if !admin_areas.iter().any(|a| a.id == parent_id) {
            return false;
        }
    }

    true
}
```

### 4.3 着色系统 (`client/src/map/coloring.rs`)

```rust
use bevy::prelude::*;
use bevy::render::mesh::{VertexAttributeValues, Indices};
use crate::map::{MapResource, MapMode, CountryMap, AdminMap, Countries, AdminAreas};
use crate::editor::AsyncTaskState;

/// 异步重新着色系统（节流处理）
pub fn recolor_mesh_system(
    mut commands: Commands,
    map: Res<MapResource>,
    country_map: Res<CountryMap>,
    admin_map: Res<AdminMap>,
    countries: Res<Countries>,
    admin_areas: Res<AdminAreas>,
    mode: Res<MapMode>,
    mut async_state: ResMut<AsyncTaskState>,
    // 查询现有的网格实体
    mesh_query: Query<(Entity, &Handle<Mesh>, &Handle<ColorMaterial>)>,
    mut meshes: ResMut<Assets<Mesh>>,
    mut materials: ResMut<Assets<ColorMaterial>>,
) {
    // 只在政治模式下重新着色
    if *mode != MapMode::Political {
        return;
    }

    // 检查是否需要重新着色
    if !async_state.pending_recolor {
        return;
    }

    // 计算每个省份的颜色
    let province_colors = compute_province_colors(
        &map, &country_map, &admin_map, &countries, &admin_areas,
    );

    // 更新网格顶点颜色
    for (entity, mesh_handle, _mat_handle) in mesh_query.iter() {
        let Some(mesh) = meshes.get_mut(mesh_handle) else { continue };

        update_mesh_colors(mesh, &province_colors);

        // 标记需要重新上传到 GPU
        // Bevy 0.15 会自动处理
    }

    async_state.pending_recolor = false;
}

/// 计算所有省份的颜色（支持行政区嵌套）
fn compute_province_colors(
    map: &MapResource,
    country_map: &CountryMap,
    admin_map: &AdminMap,
    countries: &Countries,
    admin_areas: &AdminAreas,
) -> std::collections::HashMap<ProvinceId, [f32; 4]> {
    let mut colors = std::collections::HashMap::new();

    // 构建快速查找表
    let country_colors: std::collections::HashMap<&str, [f32; 4]> = countries.0.iter()
        .map(|c| (c.tag.as_str(), c.color))
        .collect();

    let admin_colors: std::collections::HashMap<AdminId, [f32; 4]> = admin_areas.0.iter()
        .filter_map(|a| a.color.map(|c| (a.id, c)))
        .collect();

    for prov in &map.0.provinces {
        let color = get_province_color(
            prov.id,
            &country_map,
            &admin_map,
            &country_colors,
            &admin_colors,
            &admin_areas,
        );
        colors.insert(prov.id, color);
    }

    colors
}

/// 获取单个省份的颜色（优先级：行政区 > 国家 > 默认）
fn get_province_color(
    province_id: ProvinceId,
    country_map: &CountryMap,
    admin_map: &AdminMap,
    country_colors: &std::collections::HashMap<&str, [f32; 4]>,
    admin_colors: &std::collections::HashMap<AdminId, [f32; 4]>,
    admin_areas: &AdminAreas,
) -> [f32; 4] {
    // 1. 检查是否有行政区归属
    if let Some(&admin_id) = admin_map.0.get(&province_id) {
        // 尝试获取行政区颜色（向上继承）
        if let Some(&color) = get_admin_color_inherited(admin_id, admin_colors, admin_areas) {
            return color;
        }

        // 行政区无颜色，尝试获取所属国家的颜色
        if let Some(admin) = admin_areas.0.iter().find(|a| a.id == admin_id) {
            if let Some(&color) = country_colors.get(admin.country_tag.as_str()) {
                return color;
            }
        }
    }

    // 2. 检查是否有国家归属
    if let Some(country_tag) = country_map.0.get(&province_id) {
        if let Some(&color) = country_colors.get(country_tag.as_str()) {
            return color;
        }
    }

    // 3. 默认颜色（无主省份）
    [0.55, 0.55, 0.55, 1.0]
}

/// 获取行政区颜色（向上继承至国家）
fn get_admin_color_inherited(
    admin_id: AdminId,
    admin_colors: &std::collections::HashMap<AdminId, [f32; 4]>,
    admin_areas: &AdminAreas,
) -> Option<[f32; 4]> {
    let mut current_id = Some(admin_id);

    while let Some(id) = current_id {
        if let Some(&color) = admin_colors.get(&id) {
            return Some(color);
        }

        // 向上查找父行政区
        current_id = admin_areas.0.iter()
            .find(|a| a.id == id)
            .and_then(|a| a.parent_id);
    }

    None
}

/// 更新网格顶点颜色
fn update_mesh_colors(
    mesh: &mut Mesh,
    province_colors: &std::collections::HashMap<ProvinceId, [f32; 4]>,
) {
    // 获取当前的颜色属性
    let Some(VertexAttributeValues::Float32x4(existing_colors)) =
        mesh.attribute_mut(Mesh::ATTRIBUTE_COLOR)
    else {
        return;
    };

    // 更新颜色（需要根据实际的顶点布局来映射）
    // 这里假设每个省份的顶点在 mesh 中是连续的
    // 实际实现需要参考 MapPlugin 中的顶点映射逻辑

    for (i, color) in existing_colors.iter_mut().enumerate() {
        // 根据顶点索引找到对应的省份
        // 这需要在 MapPlugin 中维护 province_id → vertex_range 的映射
        if let Some(&prov_color) = province_colors.get(&(i as u32)) {
            *color = prov_color;
        }
    }
}
```

```rust
    admin_areas: Res<AdminAreas>,
    country_map: Res<CountryMap>,
    admin_map: Res<AdminMap>,
) -> io::Result<()> {
    let file = ColoringFile {
        countries: countries.0.clone(),
        country_assignments: country_map.0.clone(),
        admin_areas: admin_areas.0.clone(),
        admin_assignments: admin_map.0.clone(),
    };

    let json = serde_json::to_string_pretty(&file)
        .map_err(|e| io::Error::new(io::ErrorKind::InvalidData, e))?;

    fs::write(COLORING_FILE, json)?;
    eprintln!("已保存着色数据到 {}", COLORING_FILE);
    Ok(())
}

/// 从 JSON 文件加载着色数据
pub fn load_coloring(
    mut commands: Commands,
) {
    let Ok(json) = fs::read_to_string(COLORING_FILE) else {
        eprintln!("未找到着色文件，使用默认数据");
        return;
    };

    let file: ColoringFile = match serde_json::from_str(&json) {
        Ok(f) => f,
        Err(e) => {
            eprintln!("解析着色文件失败：{}", e);
            return;
        }
    };

    commands.insert_resource(Countries(file.countries));
    commands.insert_resource(AdminAreas(file.admin_areas));
    commands.insert_resource(CountryMap(file.country_assignments));
    commands.insert_resource(AdminMap(file.admin_assignments));

    eprintln!("已加载着色数据从 {}", COLORING_FILE);
}
```

```rust
use std::time::{Duration, Instant};

#[derive(Resource)]
pub struct RecolorTimer {
    pub last_recolor: Instant,
    pub cooldown: Duration,
}

impl Default for RecolorTimer {
    fn default() -> Self {
        Self {
            last_recolor: Instant::now() - Duration::from_secs(1), // 允许立即触发
            cooldown: Duration::from_millis(100), // 100ms 冷却
        }
    }
}

pub fn throttled_recolor_system(
    mut timer: ResMut<RecolorTimer>,
    // ...
) {
    if timer.last_recolor.elapsed() < timer.cooldown {
        return; // 跳过本次请求
    }

    // 执行重新着色
    timer.last_recolor = Instant::now();
}
```

## 8. 技术难点与解决方案

### 难点 1：行政区嵌套验证
**问题**：确保省份只能被添加到父行政区的省份集合中。

**解决方案**：
- 在 `validate_admin_assignments` 系统中进行实时验证
- 使用 BFS/DFS 遍历行政区树验证层级关系
- 无效分配自动移除并提示用户

### 难点 2：丝滑的刷子体验
**问题**：大量省份的实时着色会阻塞 UI。

**解决方案**：
- 使用 `AsyncComputeTaskPool` 异步处理
- 实现 100ms 节流机制
- 增量更新只处理变化的省份
- 使用顶点缓冲区部分更新

### 难点 3：行政区颜色继承
**问题**：无限层级的颜色继承逻辑复杂。

**解决方案**：
- 使用迭代而非递归避免栈溢出
- 缓存每个行政区的最终颜色（含继承）
- 颜色变更时只更新受影响的子树

### 难点 4：大规模网格性能
**问题**：22,000+ 省份的网格更新开销大。

**解决方案**：
- 使用单一合并网格减少 draw call
- 按省份分段存储顶点便于局部更新
- 使用 GPU instancing 渲染重复元素
- LOD（Level of Detail）根据缩放级别简化
