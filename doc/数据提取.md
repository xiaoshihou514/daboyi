# 数据提取指南

本文档描述从 EU5（Europa Universalis V）原始数据到 daboyi 游戏资产的完整处理流程，确保数据提取可复现。

---

## 目录结构约定

```
~/Playground/github/
├── EU5toGIS/          # GIS 数据仓库（EU5 地理数据集）
│   ├── datasets/
│   │   ├── locations.gpkg                  # 省份多边形矢量数据（主数据源）
│   │   ├── ports.gpkg                      # 港口位置与海区连接
│   │   ├── rivers.tif                      # 河流栅格（Gall 球极投影）
│   │   ├── rivers.png                      # EU5 原始河流调色板图像 ← 需手动复制
│   │   └── ...
│   └── localization/
│       └── simp_chinese/
│           ├── country_names_l_simp_chinese.yml   ← 需手动复制
│           └── location_names/                    ← 需手动复制（62个文件）
│               ├── location_names_l_simp_chinese.yml
│               ├── location_names_mandarin_chinese_l_simp_chinese.yml
│               └── ...
└── daboyi/            # 本游戏仓库
    ├── tools/
    │   ├── extract_rivers_vector.py   # 河流矢量化提取
    │   ├── extract_translations.py    # 国家/省份名称提取
    │   ├── convert_rivers.py          # rivers.tif → rivers.png（备用）
    │   └── mapgen/                    # 地图二进制编译器
    └── assets/
        ├── rivers.bin          ← 由 extract_rivers_vector.py 生成
        ├── map.bin             ← 由 cargo run -p mapgen 生成
        ├── terrain.bin         ← 由 cargo run -p mapgen 生成
        ├── country_names.tsv   ← 由 extract_translations.py 生成
        └── province_names.tsv  ← 由 extract_translations.py 生成
```

## 第一步：获取 EU5toGIS 数据集

EU5toGIS 是由社区制作的 EU5 地理数据集，包含省份多边形和河流栅格。
- 作者：DavidRMilos
- 下载地址：（见 EU5toGIS README）

将仓库克隆至 `../EU5toGIS`（与 daboyi 同级目录）。

---

## 第二步：从 EU5 安装目录复制文件

以下文件需从 EU5 Steam 安装目录（`<EU5_ROOT>`）手动复制到 EU5toGIS，
因为这些文件不包含在 EU5toGIS 数据集中。

**Linux 默认路径：**
```
<EU5_ROOT> = ~/.local/share/Steam/steamapps/common/Europa Universalis V/game
```

### 2a. 河流调色板图像

```bash
cp "<EU5_ROOT>/in_game/map_data/rivers.png" \
   ../EU5toGIS/datasets/rivers.png
```

### 2b. 国家名称本地化

```bash
cp "<EU5_ROOT>/main_menu/localization/simp_chinese/country_names_l_simp_chinese.yml" \
   ../EU5toGIS/localization/simp_chinese/
```

### 2c. 省份名称本地化（整个目录，62个文件）

```bash
mkdir -p ../EU5toGIS/localization/simp_chinese/location_names
cp "<EU5_ROOT>/main_menu/localization/simp_chinese/location_names/." \
   ../EU5toGIS/localization/simp_chinese/location_names/
```

**一键脚本（在 daboyi 目录下运行）：**
```bash
EU5="$HOME/.local/share/Steam/steamapps/common/Europa Universalis V/game"
EU5GIS="../EU5toGIS"
cp "$EU5/in_game/map_data/rivers.png" "$EU5GIS/datasets/"
mkdir -p "$EU5GIS/localization/simp_chinese/location_names"
cp "$EU5/main_menu/localization/simp_chinese/country_names_l_simp_chinese.yml" \
   "$EU5GIS/localization/simp_chinese/"
cp -r "$EU5/main_menu/localization/simp_chinese/location_names/." \
   "$EU5GIS/localization/simp_chinese/location_names/"
```

## 第三步：生成翻译文件

从 EU5toGIS 的本地化文件提取国家和省份的中文名称：

```bash
python3 tools/extract_translations.py
```

**输出：**
- `assets/country_names.tsv`：约 2,391 个国家标签 → 中文名称（简体）
- `assets/province_names.tsv`：约 29,492 个省份标签 → 中文名称

**加载优先级（省份名称）：**
1. 语言特定文件（最低，如 arabic、armenian 等）
2. 主 `location_names_l_simp_chinese.yml`
3. `location_names_mandarin_chinese_l_simp_chinese.yml`（最高，适用于中国/西藏地区）

---

## 第四步：生成河流矢量数据

从 EU5 河流调色板图像提取矢量折线：

```bash
python3 tools/extract_rivers_vector.py
```

**依赖：** `pip install Pillow numpy scikit-image scipy`

**处理流程：**
1. 读取 `EU5toGIS/datasets/rivers.png`（16384×8192，调色板索引图）
2. 提取河流像素（调色板索引 3–15 = 河流，索引 0 = 陆地背景）
3. 最大值池化下采样至 8192×4096（保持河流连通性）
4. 形态学闭运算（填补 1 像素断口）
5. 骨架化（细化至 1 像素宽）
6. 图边缘路径提取（在交汇点/端点间追踪每段河流）
7. Douglas-Peucker 简化（epsilon=0.5 像素）
8. 像素坐标 → WGS84 经纬度（逆 Gall 球极投影公式）
9. 写入 `assets/rivers.bin`（bincode 格式，约 600KB）

**输出格式：**
```
RiverData {
    rivers: Vec<RiverPolyline {
        points: Vec<[f32; 2]>,   // (经度, 纬度) WGS84
        width_class: u8,         // 0=细支流, 1=中等, 2=大河
    }>
}
```

---

## 第五步：编译地图二进制

从 `EU5toGIS/datasets/locations.gpkg` 和 `ports.gpkg` 生成地图数据：

```bash
cargo run -p mapgen
```

**输出：**
- `assets/map.bin`（~102MB，22,688 个省份多边形 + 元数据，已被 .gitignore 排除）
- `assets/terrain.bin`（~41MB，7,462 个水域/荒地多边形）

**注意：** `map.bin` 超过 GitHub 100MB 限制，因此被 .gitignore 排除。
每次克隆仓库后需重新运行 `cargo run -p mapgen` 生成。

## 完整重建流程（新机器）

```bash
# 1. 克隆仓库
git clone https://github.com/xiaoshihou514/daboyi
cd daboyi

# 2. 复制 EU5 文件到 EU5toGIS（见第二步）
# （需要 EU5 游戏本体和 EU5toGIS 数据集）

# 3. 生成翻译 TSV
python3 tools/extract_translations.py

# 4. 生成河流矢量数据
pip install Pillow numpy scikit-image scipy
python3 tools/extract_rivers_vector.py

# 5. 编译地图二进制
cargo run -p mapgen

# 6. 启动服务器和客户端
cargo run -p server &
cargo run -p client
```
